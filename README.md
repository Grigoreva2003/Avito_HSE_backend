## Запуск тестов
Активируй виртуальное окружение
`
source .venv/bin/activate
`

Установи зависимости (если ещё не установлены)
`pip install -r requirements.txt`

Запусти приложение (опционально, для локальной проверки)
`python3 main.py`

Запусти тесты
`python -m pytest tests/test_ads.py -v`


**Примечания**
Тесты используют FastAPI TestClient, поэтому сервер поднимать не обязательно.
Достаточно корректно настроенного приложения и импортируемого объекта app в main.py.
